NOTE:
    Neu delay_time cang TO  => chuong trinh chay cang cham
    delay_time TO khi FRAME_PER_SECOND cang nho
    vi FPS cang nho => timer_one_frame cang to => time_one_frame - real_time cang to
    
=> FPS cang nho => chuong trinh chay cang cham
*De chuong trinh chay nhanh => tang FPS
Vi du:
        fps = 25 => time_one_frame = 40;
        real_time = 27;
        =>  delay_time = 13;

*Khi fps = 38, 40, 45...
     khi fps = 37,1 => time_one_frame =27
                    va real_time = 27
     => real_time >= time_one_frame
     => delay_time = time_one_frame - real_time <= 0
     => do tre = 0; chuong trinh dat toi chat luong toi da
*De fps vuot qua gioi han la 37:
  real_time phai nho hon nua
  real_time = 10 => fps = 1000/10 = 1000;
  real_time = 5 => fps = 1000/5 = 200;
Vay: real_time cang nho khi cau hinh may tinh cang tot
**khi may tinh co cau hinh cang cao:
=> chuong trinh chay cang nhanh
